HTML

Struktur Dasar Dokumen HTML

Setiap halaman website dimulai dari kerangka dokumen HTML yang terdiri dari deklarasi tipe dokumen, elemen root, bagian metadata (<head>), dan bagian konten tampil (<body>).

<!-- Deklarasi Tipe Dokumen HTML5 --> <!DOCTYPE html> <html lang="id"> <head> <meta charset="utf-8" /> <meta name="viewport" content="width=device-width, initial-scale=1" /> <title>Judul Halaman</title> <link rel="stylesheet" href="style.css" /> </head> <body> <!-- Konten website ditampilkan di sini --> </body> </html>

🏗️ Anatomi Dokumen

  • <!DOCTYPE html> — Deklarasi bahwa dokumen ini adalah HTML5
  • <html lang="id"> — Elemen akar (root) dengan atribut bahasa
  • <head> — Metadata: judul, charset, viewport, tautan CSS
  • <body> — Seluruh konten yang tampak di layar browser

📌 Poin Penting

  • meta viewport wajib untuk website responsif
  • charset utf-8 mendukung karakter Bahasa Indonesia
  • Satu file .html = satu halaman website
HTML

Elemen Semantik & Pemformatan Teks

🧱 Elemen Semantik HTML5

Elemen yang memberikan makna pada struktur halaman, bukan hanya tampilan:

  • <header> — Bagian kepala/banner halaman
  • <nav> — Kelompok tautan navigasi
  • <main> — Konten utama halaman (hanya 1×)
  • <section> — Bagian tematik dari konten
  • <article> — Konten mandiri (blog post, berita)
  • <aside> — Konten samping/pendukung
  • <footer> — Bagian kaki halaman

✏️ Pemformatan Teks

  • <h1> — <h6> — Heading (judul) level 1–6
  • <p> — Paragraf teks
  • <strong> / <b> — Teks tebal (penting / visual)
  • <em> / <i> — Teks miring (penekanan / visual)
  • <br> — Pindah baris paksa
  • <hr> — Garis pemisah horizontal
  • <small> — Teks berukuran kecil

🔗 Tautan & Gambar

  • <a href="url"> — Hyperlink antar halaman
  • <img src="file" alt="..."> — Menyisipkan gambar
HTML

Tabel, Daftar, & Formulir

📊 Tabel Data

  • <table> — Wadah tabel utama
  • <thead> / <tbody> — Kelompok header / isi
  • <tr> — Baris tabel
  • <th> — Sel judul kolom (header cell)
  • <td> — Sel data biasa

📋 Daftar (List)

  • <ul> — Daftar tanpa urutan (bullet)
  • <ol> — Daftar berurutan (nomor)
  • <li> — Item dalam daftar

📝 Formulir (Form)

  • <form> — Wadah formulir
  • <input type="text"> — Kotak masukan teks
  • <input type="email"> — Masukan email
  • <input type="password"> — Masukan sandi
  • <textarea> — Area teks multi-baris
  • <select> + <option> — Dropdown pilihan
  • <button> — Tombol aksi
  • <label> — Keterangan input (untuk A11y)

CSS

Selektor, Properti, & Box Model

🎯 Jenis Selektor CSS

  • Element — h1 { } seleksi berdasarkan tag
  • Class — .card { } seleksi berdasarkan kelas (titik)
  • ID — #header { } seleksi unik berdasarkan id (pagar)
  • Grouping — h1, h2, h3 { } mengelompokkan aturan
  • Descendant — .card p { } seleksi anak di dalam induk

🎨 Properti Umum

  • color — Warna teks
  • background-color — Warna latar belakang
  • font-family — Jenis huruf
  • font-size — Ukuran huruf
  • font-weight — Ketebalan huruf (bold)
  • text-align — Perataan teks

📦 Box Model

Setiap elemen HTML membentuk sebuah "kotak" yang terdiri dari 4 lapisan konsentris:

MARGIN
BORDER
PADDING
CONTENT
CSS

Flexbox & CSS Grid

📐 Flexbox (Tata Letak 1 Dimensi)

Mengatur elemen di sepanjang satu sumbu (baris ATAU kolom):

  • display: flex — Mengaktifkan Flexbox pada induk
  • flex-direction — row | column (arah sumbu utama)
  • justify-content — Perataan sepanjang sumbu utama
  • align-items — Perataan sepanjang sumbu silang
  • gap — Jarak antar item flex
  • flex-wrap — Izinkan pelipatan baris otomatis

🔲 CSS Grid (Tata Letak 2 Dimensi)

Mengatur elemen di sepanjang dua sumbu sekaligus (baris DAN kolom):

  • display: grid — Mengaktifkan Grid pada induk
  • grid-template-columns — Definisi jumlah & lebar kolom
  • grid-template-rows — Definisi jumlah & tinggi baris
  • gap — Jarak antar sel grid
  • fr — Satuan fraksi fleksibel (1fr = 1 bagian)
  • repeat(3, 1fr) — 3 kolom berukuran sama

Bootstrap

Grid System & Responsive Breakpoints

🏗️ Sistem Grid 12-Kolom

Bootstrap membagi lebar halaman menjadi 12 kolom fleksibel:

  • .container — Wadah terluar (centered, responsive)
  • .row — Baris grid (membungkus kolom-kolom)
  • .col-{bp}-{n} — Kolom berukuran n/12 pada breakpoint bp
  • .col — Kolom otomatis (bagi rata)
<div class="container"> <div class="row"> <div class="col-md-8">Konten Utama</div> <div class="col-md-4">Sidebar</div> </div> </div>

Bootstrap

Komponen Advanced (Grid Lanjut, ARIA, & Sass)

🔲 Advanced Grid System

  • Nested Grid — Membuat baris di dalam kolom
  • Column Ordering — .order-md-* mengubah urutan visual
  • Gutters Control — .g-*, .gx-*, .gy-* jarak antar kolom
  • Column Breaks — .w-100 paksa pindah baris
  • Vertical/Horizontal Alignment — perataan sumbu grid

♿ Aksesibilitas (ARIA)

  • aria-label — Deskripsi elemen untuk screen reader
  • aria-expanded — Status buka/tutup komponen
  • aria-controls — ID elemen yang dikontrol
  • role — Peran semantik elemen (dialog, alert, navigation)

🎨 Sass (SCSS) & Kustomisasi

  • Variables — Menimpa $primary, $success
  • Nesting — CSS bertingkat dalam induk
  • Mixins — Fungsi gaya yang reusable
  • Compile — npm run build-css
Bootstrap

Peserta didik mengintegrasikan seluruh konsep HTML, CSS, dan Bootstrap untuk mengembangkan website statis responsif berdasarkan tema tertentu seperti profil pribadi, company profile, atau website sekolah.

Persyaratan Teknis Minimum

✅ Wajib Dipenuhi (Minimum Requirements)

  • ≥ 3 halaman HTML yang saling terhubung melalui navigasi
  • Navbar responsif menggunakan Bootstrap (collapse pada mobile)
  • Bootstrap Grid System — minimal menggunakan .container, .row, .col-*
  • ≥ 3 Komponen Bootstrap berbeda (contoh: Card, Alert, Table, Accordion, Modal, Carousel, dsb.)
  • File CSS terpisah (style.css) untuk kustomisasi tambahan di luar Bootstrap
  • Minimal 1 gambar yang relevan dengan tema
  • Footer konsisten di semua halaman
  • Website tampil responsif saat diakses dari layar ponsel (≤768px)

⭐ Nilai Bonus (Opsional)

  • Menggunakan Google Fonts kustom
  • Menambahkan efek hover / transisi CSS
  • Menggunakan Carousel / Modal / Toast
  • Menerapkan atribut ARIA untuk aksesibilitas
  • Menggunakan Sass (SCSS) untuk kustomisasi tema
  • Desain yang sangat kreatif & wow factor tinggi

  • Gunakan tanda hubung (-) sebagai pemisah kata, bukan spasi

  • Simpan semua gambar di dalam folder images/

⚠️ Kesalahan Umum

  • ❌ Nama file dengan spasi: Halaman 1.html
  • ❌ Tidak konsisten: Index.HTML vs style.CSS
  • ❌ Gambar ditaruh di luar folder: berserakan

Tips & Best Practices Pengerjaan

💡 Mulai dari Struktur, Bukan Desain

Buat kerangka HTML lengkap di ketiga halaman terlebih dahulu (header, konten, footer). Baru kemudian percantik tampilannya dengan CSS & Bootstrap.

🔄 Copy-Paste Navbar ke Semua Halaman

Buat navbar di halaman pertama (index.html), lalu salin ke halaman 2 dan 3. Ubah kelas active sesuai halaman yang sedang dibuka.

📱 Tes Responsif Secara Berkala

Tekan F12 di browser → klik ikon perangkat mobile → pilih resolusi "iPhone SE" atau "iPad". Pastikan tidak ada teks atau gambar yang terpotong.

🎨 Jangan Terlalu Banyak Warna

Pilih maksimal 2–3 warna utama dan gunakan secara konsisten di semua halaman. Gunakan variasi shade warna Bootstrap yang sudah disediakan.

🚫 Hindari Kesalahan Fatal

  • Lupa memasang CDN Bootstrap di <head>
  • Tautan navigasi rusak (href salah ketik)
  • Gambar tidak muncul (path file salah)
  • Tidak menggunakan .container sebagai wadah

📋 Cek Ulang Sebelum Kumpul

  • Buka setiap halaman, klik semua link menu
  • Kecilkan layar browser ke ukuran ponsel
  • Pastikan gambar muncul semua
  • Pastikan footer ada di semua halaman

Template Awal: Kerangka Kode HTML + Bootstrap

Gunakan template berikut sebagai titik awal. Salin ke setiap file HTML, lalu modifikasi kontennya sesuai kebutuhan masing-masing halaman:

<!DOCTYPE html> <html lang="id"> <head> <meta charset="utf-8" /> <meta name="viewport" content="width=device-width, initial-scale=1" /> <title>Judul Halaman</title> <!-- Bootstrap 5 CSS CDN --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" /> <!-- CSS Kustom --> <link rel="stylesheet" href="style.css" /> </head> <body> <!-- NAVBAR (salin ke semua halaman) --> <nav class="navbar navbar-expand-lg navbar-dark bg-primary sticky-top"> <div class="container"> <a class="navbar-brand fw-bold" href="index.html">Nama Website</a> <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navMenu"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="navMenu"> <ul class="navbar-nav ms-auto"> <li class="nav-item"> <a class="nav-link active" href="index.html">Beranda</a> </li> <li class="nav-item"> <a class="nav-link" href="halaman-2.html">Halaman 2</a> </li> <li class="nav-item"> <a class="nav-link" href="halaman-3.html">Halaman 3</a> </li> </ul> </div> </div> </nav> <!-- KONTEN UTAMA --> <main class="container my-5"> <h1>Selamat Datang</h1> <!-- Isi konten halaman di sini --> </main> <!-- FOOTER (salin ke semua halaman) --> <footer class="bg-dark text-white text-center py-4 mt-5"> <p class="mb-0">&copy; 2026 Nama Siswa — Kelas X</p> </footer> <!-- Bootstrap 5 JS CDN --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
