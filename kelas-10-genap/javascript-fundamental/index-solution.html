5. Applying Computation (Kalkulator Harga)
Lengkapi fungsi perkalian otomatis di JavaScript. Masukkan kuantitas pembelian buku, harga per buku konstan didefinisikan sebesar Rp 50.000, lalu hitung totalnya.
*Perhitungan dilakukan secara asinkron/langsung melalui event handler input.
6. Playground Eksperimen Variabel (var, let, const)
Silahkan buka Developer Tools di browsermu dengan menekan tombol Ctrl + Shift + J (Windows/Linux) atau Cmd + Option + C (Mac). Tuliskan skrip pengujian variabel pada tag <script> di bawah dan perhatikan hasil log atau pesan eror yang muncul.
  • Eksperimen 1: Deklarasikan ulang variabel var, cek apakah nilainya terupdate tanpa eror.
  • Eksperimen 2: Coba deklarasikan ulang variabel let, amati pesan eror SyntaxError.
  • Eksperimen 3: Coba ubah nilai variabel const, amati pesan eror TypeError.
  • Eksperimen 4: Uji keluaran dialog menggunakan window.alert() dan tampilkan pesan di konsol menggunakan console.log().
